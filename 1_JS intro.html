Button 5 is the same as button 4 but with querySelector. The querySelectorAll provides more oportunity than getElementsByTagName. querySelectorAll gives back a node list but getElementsByTagName gives back only an HTML collection. Check the difference in the console.

querySelectorAll can accept every existing css selector. Eg.: querySelectorAll(".my-class:first-child")

querySelector gives back only one element. If there are more elements it gives back the first one.
