.forEach()

Calls a function for each element in an array. Not for empty elements.

Return value: undefined

You can use it to create something new from the elements of an array. Eg.: to sum the elements and store it in a variable.

.map()

Creates a new array from calling the given function once for every array element. Not for empty elements. Does not change the original array.

Return value: array (of the changed elements)
